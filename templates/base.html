Welcome !

This web application can help you recognise your favourite anime characters.

There are 2 models for recognition. Model 1 seperates all the faces in the image given by user and then gives the name and details of each character.

Whereas model 2 is for images which have only a single charcater in them. It gives a single prediction according to the image uploaded by the user.

Note: Sometimes if the given image is unclear, model 1 fails to recognise faces and does not give a prediction. In this case you can use model 2 to get a prediction.


Also, Please go through the README section of the website to check the list of supported Anime characters and other details about this website.

Welcome ! This web application can help you recognise your favourite anime characters.

There are 2 models for recognition. Model 1 seperates all the faces in the image given by user and then gives the name and details of each character. Whereas model 2 is recommended if there is only one character in the image, as the model is not only trained on the face features but overall features of the character.

Note: Sometimes if the given image is unclear, model 1 fails to recognise faces and does not give a prediction. In this case you can manually crop the image and use model 2 to get a prediction.
Also, Please go through the README section of the website to check the list of supported Anime characters and other details about this website.
